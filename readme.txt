Add a mobile friendly fluid width Boxed Content with an easy to use shortcodes

== Description ==

Responsive Boxed Content Shortcodes 

This easy to use boxed content shortcodes generates an fluid width boxed content that maximizes the width of the container.

Options you can set are:
- Color
- Height
- Opacity

Please see Frequently asked questions for shortcode usage.

This plugin was created as part of the [Marketer WordPress business theme](http://themersclub.com/marketer/shortcodes/box)

== Installation ==

1. Upload `boxed-content` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add via shortcode

== Frequently asked questions ==

= How do I use the shortcodes? =

Here is the shortcode usage:
`[box color="#CCC" height="200px" opacity="0.9"]`

Color, height and opacity are optional attributes
